# Roadmap dr.Climate CRM

## Сделано (текущий релиз)

- Брендинг **dr.Climate**
- Главная: логотип + вход / регистрация
- Необязательная стоимость при создании заявки (админ)
- Управление городами: редактирование, деактивация

## Следующий этап

1. **Редактирование заявки админом** — цена, адрес, статус
2. **Push / звук** — уведомление мастера о новой заявке
3. **Список мастеров у админа** — онлайн-статус, встроенный чат с мастером
4. **Аудит действий** — кто создал / изменил заявку
5. **PWA** — установка на телефон для кабинета мастера
