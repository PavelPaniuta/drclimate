# Roadmap dr.Climate CRM

## Сделано

- Брендинг **dr.Climate**, главная с входом/регистрацией
- Стоимость при создании заявки (необязательно)
- Управление городами (редактирование, деактивация)
- **Редактирование заявки админом** (цена, адрес, статус) + история аудита
- **Звук и push** при новой заявке у мастера (WebSocket + Notification API)
- **Список мастеров у админа** с онлайн-статусом и **чат админ ↔ мастер**
- **Аудит** действий по заявкам
- **PWA** для кабинета мастера (`manifest.webmanifest`, service worker)

## Возможные улучшения

- Нативные push через FCM/Web Push с VAPID
- Редактирование заявки: описание, тип услуги, переназначение мастера
- Непрочитанные сообщения в чате (badge)
- Установочный баннер PWA на iOS
